## ✨ Highlights

*   **Dark/Light Mode Toggle**: Built-in CSS variables allow for seamless theme switching.
*   **3D Hero Animation**: Neural network interactive visualization powered by `Three.js`.
*   **AI Career Path Finder**: A robust 5-step interactive quiz to help students find their ideal course track based on goals and background.
*   **Interactive Skills Map**: A custom-built HTML5 Canvas visualization for exploring technology stacks and tools.
*   **Real-time Job Readiness Meter**: Dynamic demand meters displaying market trends, top hiring companies, and skill gaps for different IT domains.
*   **Mock Dashboard LMS**: A preview of the student portal showcasing course progress, coding simulator, and certifications.
*   **LiveBot AI Assistant**: Built-in interactive admissions chatbot with quick replies and dynamic typographics.
*   **Smart Enrollment Flow**: A multi-step stepper guiding students from course selection to batch configuration seamlessly.

---

## 🛠 Tech Stack

- **HTML5**: Semantic tags, accessible structure, structured for optimal SEO.
- **Vanilla CSS (CSS3)**: Extensive custom styling natively utilizing CSS Variables, Animations, Flexbox, CSS Grid, and responsive media queries.
- **Vanilla JavaScript**: DOM Manipulation, complex interactions, intersection observers, and logic driving the AI Path Finder.
- **Three.js**: Lightweight 3D scene rendering in the Hero section.

## 🚀 Quick Start

1. **Clone the repository** to your local machine:
   ```bash
   git clone https://github.com/your-username/livewire-kasaragod.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd livewire-kasaragod
   ```

3. **Open the project**:
   Since this project uses vanilla web technologies without any build steps, simply double-click the `livewire-kasaragod-v2.html` file to open it in your browser. Alternatively, use tools like **Live Server** (VS Code Plugin) for hot-reloading.

## 📝 Features & Modules Breakdown

### 1. Hero Pipeline
Dual-themed 3D scene that rotates and scales dynamically with mouse events to immediately immerse the user into a tech environment.

### 2. Career Path Finder
Tracks user selections across five domains (Interest, Skill Level, Career Goal, Time Commitment, and Background) to logically route them to one of five comprehensive career path roadmaps.

### 3. Dashboard Portal
Mock environment demonstrating what students experience after enrolling. Features include real-time progress bars, syntax-highlighted code mockups, and certificate previews.

### 4. Interactive Chat (LiveBot) 
Simulated AI chatbot answering standard queries regarding placement, fees, syllabus, and course duration with organic typing delays.
